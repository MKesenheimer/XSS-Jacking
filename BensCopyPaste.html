Awareness

Viele kopieren gerne Code von Webseiten. Den liest man im Idealfall vorher schon mal - aber schaut ihr euch den auch an, wenn ihr ihn in euer Terminal gepastet habt? Denn manchmal gibt es ein Überraschungs-Ei. Viel Spaß bei Copy&Paste des folgenden harmlosen Commandos (kann man wirklich mal ausführen, es passiert nichts schlimmes. Versprochen):

sudo id && echo 'WTF!?' &&whoami
